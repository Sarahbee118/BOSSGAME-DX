Install Guide:

Android - Google Play Version:

0. Purchase and Download BOSSGAME from the Google Play Store (https://play.google.com/store/apps/details?id=com.LilycoreGames.Bossgame&hl=en_US&gl=US)
1. Install Unipatcher (https://play.google.com/store/apps/details?id=org.emunix.unipatcher&hl=en_US&gl=US)
2. Install and Open Apk Extractor (https://play.google.com/store/apps/details?id=com.ext.ui&hl=en_US&gl=US)
2.5 If you can't install Apk Extractor from the Play Store then download it from https://m.apkpure.com/apk-extractor/com.ext.ui
3. Scroll down and select Bossgame (you should see "Extracting" for a moment)
4. Open Unipatcher
5. For Patch File select PlayStorePatch.xdelta from this folder
6. For ROM select Bossgame_base.apk from /ExtractedApks/Bossgame_com.LilycoreGames.Bossgame
7. For Output File save the patched APK somewhere easy to access
8. Hit the button in the bottom right of the app
9. Once patched, located the patched APK with a file manager (you'll know it's patched by the rainbow icon)
10. Select it to install it, allow from unknown sources if needed
